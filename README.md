# Ignis — Interactive Volcano Map

An interactive map of the world's active volcanoes, powered by daily USGS data.

**Live:** https://simonestella.github.io/ignis/

## Stack

- **Next.js 15** — App Router, static export
- **React Leaflet** — interactive map
- **Framer Motion** — animations
- **Tailwind CSS v4** — styling
- **next-themes** — light/dark mode
- **USGS GVP API** — volcano data, cached daily in localStorage
